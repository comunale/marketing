# Playbook do Cientista do Marketing  
## Introdução e Regras de Uso

## O Que é Este Playbook

Este playbook é um **guia de decisão**.

Ele existe para orientar:
- o que fazer
- o que priorizar
- o que NÃO fazer
- quando agir
- quando parar

Ele não é material educativo.
Ele não é livro.
Ele não é curso.

Ele é a tradução prática do método do Cientista do Marketing
em **regras operacionais aplicáveis no dia a dia**.

---

## Para Quem Este Playbook Existe

Este playbook foi criado para:
- gestores de marketing
- líderes de crescimento
- consultores
- equipes internas
- agentes de IA responsáveis por recomendar ações

Se você busca inspiração ou motivação, este não é o material.
Se você busca **clareza e decisão**, este é.

---

## O Papel Deste Playbook no Sistema

Este playbook ocupa uma posição específica:

- A **base de conhecimento** explica o *porquê*
- O **playbook** define o *como agir*
- A **execução** valida ou corrige decisões

O playbook nunca substitui análise,
mas elimina decisões improvisadas.

---

## Como Usar Este Playbook (Humanos)

Sempre use este playbook da seguinte forma:

1. Identifique o problema real  
2. Localize o bloco correspondente  
3. Aplique as regras descritas  
4. Execute com disciplina  
5. Meça o impacto  
6. Volte ao diagnóstico se necessário  

Não pule etapas.
Não adapte regras por conveniência.
Não misture intuição com exceção.

---

## Como Usar Este Playbook (IA)

Quando usado por sistemas de IA, este playbook deve:

- ter prioridade sobre conteúdos teóricos
- ser usado para responder perguntas do tipo:
  - “o que fazer agora?”
  - “o que priorizar?”
  - “qual o próximo passo?”
- orientar respostas diretas e acionáveis
- evitar recomendações genéricas

Se houver conflito entre:
- opinião
- criatividade
- preferência pessoal

o playbook prevalece.

---

## O Que Este Playbook NÃO Faz

Este playbook não:
- cria campanhas
- escreve anúncios
- define criativos
- substitui análise de contexto
- justifica decisões ruins

Ele define **critérios**.
A aplicação correta continua sendo responsabilidade de quem executa.

---

## Princípios que Regem Este Playbook

Este playbook opera sob princípios inegociáveis:

- crescimento é sistêmico
- método vem antes de ferramenta
- decisão precede execução
- retenção vem antes de escala
- dados vencem opinião
- disciplina vence improviso

Qualquer ação que viole esses princípios
deve ser questionada imediatamente.

---

## Atualização e Evolução

Este playbook é um sistema vivo.

Ele deve ser:
- revisado periodicamente
- ajustado com base em evidência
- evoluído com aprendizado real

Mudanças sem evidência são regressão.

---

## Autoridade do Playbook

Em caso de dúvida entre:
- urgência operacional
- pressão externa
- opinião interna

este playbook deve ser consultado.

Ignorar o playbook conscientemente
é assumir responsabilidade total pelo risco.
